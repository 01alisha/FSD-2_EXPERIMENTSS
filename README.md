# React SPA Project

## Description
This project is a Single Page Application (SPA) built using React. It demonstrates client-side routing, API integration, and component-based architecture.

## Features
- Home, About, Users pages
- Client-side routing using react-router-dom
- API data fetching
- React hooks (useState, useEffect)

## Technologies Used
- React
- Vite
- JavaScript
- CSS

## Setup
npm install  
npm run dev
