# Form Handling in React

## Description
This experiment demonstrates form handling techniques in React including validation, file upload, and multi-step forms.

## Features
- Controlled components
- Form validation
- React Hook Form
- Yup validation
- File upload
- Multi-step form
- Error messages

## Technologies Used
- React
- Vite
- React Hook Form
- Yup

## Setup
npm install  
npm run dev
