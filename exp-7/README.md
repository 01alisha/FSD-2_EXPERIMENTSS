# API Integration in React

## Description
This experiment demonstrates API integration using Axios in React.

## Features
- Fetch data using GET request
- Add data using POST request
- Loading and error handling
- Error Boundary implementation
- Authentication token concept
- CRUD operations basics

## Technologies Used
- React
- Vite
- Axios

## Setup
npm install  
npm run dev
