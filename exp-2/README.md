# UI using Component Libraries

## Description
This experiment demonstrates UI design using Material UI and Bootstrap in React.

## Features
- Material UI components (Button, Grid, Paper, TextField)
- Bootstrap integration
- Responsive layout using grid system
- Form with validation
- Dashboard UI

## Technologies Used
- React
- Vite
- Material UI
- Bootstrap
- JavaScript
- CSS

## Setup Instructions
1. npm install  
2. npm run dev
