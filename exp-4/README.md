# State Management in React

## Description
This experiment demonstrates state management using Context API and Redux Toolkit in React.

## Features
- Context API for global state
- Redux Toolkit setup
- Actions and reducers
- Global store configuration
- useSelector and useDispatch hooks
- Async actions using Redux Thunk
- Redux DevTools support

## Technologies Used
- React
- Vite
- Redux Toolkit
- React Redux

## Setup
npm install  
npm run dev
